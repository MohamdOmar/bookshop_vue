# @capire/bookshop

This is our primer sample for [Getting Started in a Nutshell](https://cap.cloud.sap/docs/get-started/in-a-nutshell).


### Get it

  ```sh
  git clone https://github.com/capire/bookshop
  ```

### Run  [^1]

  ```sh
  cds watch bookshop
  ```

### Reuse [^2]

  ```sh
  npm add @capire/common
  ```
